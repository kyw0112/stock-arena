## 게임별 경제

### 가위바위보 (RPS)
- **배당**: 1.97배 (승리 시 배팅금의 97% 수익)
- **배팅 한도**: 보유 포인트의 90%까지
- **레이크 분배**:
  - 배팅금의 4% → 잭팟 풀
  - 배팅금의 1% → 로또 풀
- **잭팟**: 100P 이상 배팅 시 0.1% 확률로 잭팟 풀 전액 당첨 (승패 무관)
- **무승부**: 배팅금 전액 반환 (레이크 없음)

### 오목 / 체스
- **대전 배팅**: 방 생성 시 설정 (0P = 무료)
- **승리**: 양쪽 배팅금 합산(2배) 수령
- **무승부**: 각자 배팅금 환불
- **기권**: 상대방에게 배팅금 지급
- **MMR**: Elo 시스템 (K=32, 기본 1000)

### 관전 배팅 (오목/체스)
- 관전자가 진행 중인 게임의 승자를 예측하고 포인트를 걸 수 있음
- **정산**: 게임 종료 시 자동
  - 적중: 전체 풀을 적중자끼리 배팅 비율로 배분
  - 실패: 배팅금 잃음
  - 무승부/취소: 전액 환불
  - 적중자 0명: 전액 환불
- **제한**: 게임 참가자는 관전 배팅 불가, 1게임당 1회만

### 주사위 (Dice)
- 다인 제거전 방식
- 참가비 → 풀 적립 → 최종 승자가 전부 수령

### 가챠 (Gacha)
- 1일 1회 무료, 추가 스핀 500P
- 등급별 보상:
  - MISS (50%): 0P
  - SMALL (30%): 10~50P
  - MEDIUM (15%): 100~300P
  - JACKPOT (5%): 500~2,000P

### 로또 (Lotto)
- 매일 15:55 자동 추첨
- 번호 범위: 1~46
- 풀 재원: 보유세 징수금의 50% + 기타
- 당첨: 풀 ÷ 당첨자 수 (미당첨 시 이월)
- 티켓 수: 모의투자 순위별 차등 (상위 1장, 하위 5장)

### 노들 (Nordle)
- 매일 1문제 (수식 맞추기)
- 최대 6번 시도
- 오늘의 랭킹: 시도 횟수 → 완료 시각 순
- 주간 랭킹: 풀이 수 → 평균 시도 횟수 순

## 포인트 소모

### 보유세 (Wealth Tax)
- **세율**: 10%
- **시점**: 매일 15:55 (로또 추첨과 동시)
- **분배**: 징수액의 50% 소각, 50% 로또 풀에 적립
- **대상**: 승인된 일반 유저 (관리자 제외)

### 상점
- 닭대가리 배지: 300P (다른 유저에게 씌우기)
- 닭대가리 제거: 500P (자기 배지 제거)

### 포인트 선물
- 유저 간 직접 이체 (수수료 없음)

## 포인트 트랜잭션 로그

모든 포인트 변동은 `point_transactions` 테이블에 기록됨:
- `user_id`: 대상 유저
- `amount`: 변동량 (+ 증가, - 감소)
- `balance_after`: 변동 후 잔고
- `source`: 출처 (출석 체크, 가위바위보, 베팅 참여, 관리자 조정 등)
- `description`: 상세 설명
- `created_at`: 시각

관리자 패널에서 유저별/소스별 필터링 조회 가능.
